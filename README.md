# 🌟 Overview 🌟
Welcome to the Cakelicious E-commerce Website! This responsive and user-friendly website is designed to provide a delightful experience for cake lovers. Built using HTML, CSS, and Vanilla JavaScript, it offers an interactive interface for browsing cakes, placing orders, and exploring delicious collections. The website is optimized for both desktop and mobile users, ensuring a smooth experience across all devices.

# ⚙️ Technologies Used ⚙️
### 1. *🟧 HTML (HyperText Markup Language)*
HTML serves as the backbone of the website, providing the structure and layout for various elements like headings, product cards, buttons, and images.

### 2. *🟦 CSS (Cascading Style Sheets)*
CSS is utilized to style the website, enhancing its visual appeal and responsiveness with modern layouts and smooth design.

### 3. *🟨 JavaScript (Vanilla JavaScript)*
JavaScript adds interactivity and functionality to the website. This project uses Vanilla JavaScript to implement dynamic features such as:
- Adding cakes to the shopping cart
- Viewing and removing items from the cart
- Form validation for user inputs
- Interactive UI elements


# ✨ Key Features ✨
*📱 **Responsive Design**:*
- Smooth navigation across devices
- Mobile-friendly layout for better user experience

*🛒 **E-commerce Functionality**:*
- Add cakes to cart easily
- View and manage cart items

*📝 **Form Validations**:*
- Contact form validation
- Email validation for newsletter/signup

*🍰 **Product Showcase**:*
- Beautiful display of cakes and categories
- Clean UI for browsing products
